Agent Based Modelling

An agent based model, is a type of program which allows developers to create objects with the same attribute types and behaviours; while allowing them to behave independantly within the model. Notwithstanding this, the agents may also interact with eat other, agents of different types and a virtual environment.
In this context an agent, is simply a data object designed to represent a thing. The thing itself, can be real or virtual. Such as a person moving about an area or a computer game character, in a virtual world.
Through flow control programing the developer is able to control how many agents or created, when they are created and/or destroyed and how and when their behaviours are activated; including any interaction with the enviroment or other agents.
The task of this project, was to develop an agent based model which creates multiple agents and causes the to move about an environment set set number of times. Following each movement the agent this "eats" the environemnt it stops; thereby reducing the environment value and increasing it's store value by the same fixed amount or whatever is left in the enviroment whichever is less. In my model, agents coming into close proximety with each other will also equally share there stores.
